# Solar & Media campaign images

Drop photos/videos here as you get them. Suggested filenames (the page will
be built to slot these in, using these names means no code changes needed
when you add them):

- `hero-worship.jpeg`: a strong worship/congregation shot for the page's hero
  background (wide/landscape, this one matters most for first impressions).
  The filename must match exactly, including the extension (`.jpeg`, not
  `.jpg`), or the page won't find it.
- `building-exterior.jpg`: the church building.
- `generator-current.jpg`: the current generator setup (for "the situation"
  section; the noise/fuel/unreliability story lands harder with a real photo).
- `worship-service.jpg`: a service/congregation photo.
- `media-team.jpg`: the sound/media/livestream team or current equipment.
- `pastor-joe.jpg`, `ariyo.jpg`: headshots for the contact section (optional).
- `hero-video.mp4`: optional short looping video for the hero background
  instead of a static photo (10-20s, no audio needed, will be muted/autoplay).

Anything you drop with these exact names will show up automatically. Until
then the page uses a placeholder (blurred gradient/icon) so the layout still
looks intentional.
