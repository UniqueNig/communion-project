# Brand assets

Drop these files here (exact filenames matter, the site code will reference them):

- `logo.png` (or `logo.svg` if you have a vector version): your white TCC logo.
  Since it's pure white, it'll be used on dark/gradient backgrounds. In light
  mode it's automatically darkened. If you have a dedicated dark/colored
  version for light backgrounds, add it as `logo-dark.png` and it'll be used
  instead.
- `favicon.png`: optional, a square version of the logo for the browser tab icon.
